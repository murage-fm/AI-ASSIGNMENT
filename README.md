# AI-ASSIGNMENT
🚢⚽ Data Science Projects: Titanic & Football Analysis
📌 Overview
This repository contains two AI projects:
Titanic Survival Prediction
Football Data Analysis
Both projects apply data cleaning, feature engineering, exploratory data analysis, and machine learning techniques to extract insights and build predictive models.
🎯 Objectives
Clean and preprocess real-world datasets
Perform feature engineering and exploratory data analysis
Build and evaluate machine learning models (Titanic project)
Analyze football data to identify trends and performance insights
🛠️ Tools Used
Python, Pandas, NumPy, Jupyter Notebook
📂 Structure

data/        → datasets  
notebooks/   → analysis notebooks  
scripts/     → reusable code  
README.md    → project documentation
🚀 How to Run
Bash
pip install -r requirements.txt
jupyter notebook
